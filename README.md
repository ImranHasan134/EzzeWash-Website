# 🧺 EzzeWash — Premium Online Laundry Service

[![Live Demo](https://img.shields.io/badge/Live_Demo-View_Website-blue?style=for-the-badge)](https://ezzewash.vercel.app/)
[![Status](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)](#)

EzzeWash is a modern, fully responsive landing page and web application interface for Bangladesh's premier online laundry and dry-cleaning service. It features a sleek glassmorphism UI, a dynamic multi-step order wizard, and a simulated AI chatbot.

## ✨ Key Features

* 🌗 **Light & Dark Mode:** Seamless theme switching with persistent user preference saved via LocalStorage. Adaptive UI elements that automatically invert for maximum readability.
* 🛒 **Multi-Step Order Wizard:** A dynamic, 5-step interactive order process (Service ➔ Store ➔ Schedule ➔ Quantity/Promo ➔ Payment) with real-time cart and discount calculations.
* 🤖 **Bubble Bot AI Assistant:** A built-in, front-end chat widget simulation that provides instant responses to common user queries regarding tracking, pricing, and services.
* 📱 **3D App Showcase:** A highly interactive app screen gallery featuring 3D hover effects on desktop and smooth, snap-scrolling on mobile devices.
* ✨ **Advanced UI/UX Animations:** Custom CSS animations including scroll-reveals (`IntersectionObserver`), dynamic floating background bubbles, and smooth `cubic-bezier` transitions.
* 🏷️ **Promo Validation System:** Integrated JavaScript logic for validating and applying discount codes dynamically to the user's cart.
* 📐 **Fully Responsive UI:** Carefully crafted with CSS Flexbox and Grid to ensure a pixel-perfect layout across desktops, tablets, and mobile phones.

## 🛠️ Tech Stack

This project is built purely with core web technologies, ensuring maximum performance without the overhead of heavy frameworks:

* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom variables, CSS Grid/Flexbox, Glassmorphism (backdrop-filter), Keyframe animations, and Media Queries.
* **JavaScript (Vanilla):** DOM manipulation, event handling, state management for the order wizard, and chatbot simulation logic.
* **Fonts & Icons:** FontAwesome 6, Google Fonts (*Alexandria* for body, *Syne* for headings).
