## 📌 Overview

**EzzeWash** is a modern, fully responsive landing page and web application interface for Bangladesh's premier online laundry and dry-cleaning service.

It features a sleek **glassmorphism UI**, a dynamic **multi-step order wizard**, and a simulated **AI chatbot**, all built using pure web technologies for maximum performance.

---

## ✨ Key Features

### 🌗 Light & Dark Mode
- Seamless theme switching
- User preference saved via `localStorage`
- Adaptive UI elements for better readability

### 🛒 Multi-Step Order Wizard
- 5-step process:
  ```
  Service ➔ Store ➔ Schedule ➔ Quantity/Promo ➔ Payment
  ```
- Real-time cart updates
- Dynamic pricing & discount calculation

### 🤖 Bubble Bot AI Assistant
- Frontend chatbot simulation
- Instant responses for:
  - Order tracking
  - Pricing
  - Services

### 📱 3D App Showcase
- Interactive UI preview
- 3D hover effects (desktop)
- Snap scrolling (mobile)

### ✨ Advanced Animations
- Scroll reveal (`IntersectionObserver`)
- Floating bubble background
- Smooth `cubic-bezier` transitions

### 🏷️ Promo Validation System
- Apply discount codes dynamically
- Real-time validation logic using JavaScript

### 📐 Fully Responsive Design
- Built using:
  - CSS Flexbox
  - CSS Grid
- Optimized for:
  - Desktop
  - Tablet
  - Mobile

## 🚀 Quick Start

No installation needed — pure frontend project.

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/ezzewash.git
```

### 2️⃣ Navigate to Folder
```bash
cd ezzewash
```

### 3️⃣ Run Project
- Open `index.html` directly  
**OR**
- Use **Live Server (VS Code)** for best experience
